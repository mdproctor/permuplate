② Error Highlighting + Quickfix — orphan variable

String "${v1}Callable${v2}" on field of type Callable2 — ${v1} has no corresponding prefix text

IJ IntelliJ IDEA — error + intention
8
9 @PermuteDeclr(type = "${v1}Callable${v2}",
10 name = "c${i}")
11 Callable2 c2;
Permuplate: Orphan variable in type string
${v1} has no corresponding prefix in Callable2.
The text before "Callable" in the class name is empty — ${v1} would always expand to nothing.
💡 Remove ${v1} → "Callable${v2}"

Red squiggle on the orphan variable; hover shows explanation and fix preview
